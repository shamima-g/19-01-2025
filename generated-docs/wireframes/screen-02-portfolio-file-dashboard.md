## User Actions

- **Upload file**: Click "Upload" â†’ opens File Import Popup (Screen 3)
- **Re-import file**: Click "Re-import" â†’ opens File Import Popup (Screen 3) in replace mode
- **View errors**: Click "View Errors" â†’ shows validation errors in modal
- **Cancel import**: Click "Cancel" â†’ stops ongoing import job
- **Refresh status**: Click "Refresh Status" â†’ updates all status indicators
- **Proceed to next step**: Click "Proceed to Other Files" â†’ navigates to Screen 4

## Status Definitions

| Status | Icon | Meaning |
|--------|------|---------|
| Done | âœ“ | File imported and validated successfully |
| Processing | ğŸ”„ | File upload in progress or validation running |
| Pending | â³ | File not yet uploaded |
| Fix Data | âš ï¸ | File imported but has validation errors - requires fixing |

## Navigation

- **From:** Start Page (Screen 1) after creating new batch
- **To:**
  - File Import Popup (Screen 3) - via Upload/Re-import buttons
  - Other Files Dashboard (Screen 4) - via "Proceed to Other Files" button
  - Start Page (Screen 1) - via "Back to Start" button

## Notes

- Each portfolio shows all 7 file types in a consistent order
- Status updates automatically every 30 seconds if any file is "Processing"
- "Fix Data" status requires user to fix data in source system and re-import
- Row counts are shown for successful imports
- Error counts are shown for failed validations
- Portfolio cards can be collapsed to save screen space
